**灵台量化 (Quant Company)** 欢迎君至——一套多智能体量化交易网络。

此网携六员专灵，共成完整量化交易管线：

| 器灵 | 司职 |
|------|------|
| quant_lead | 协调者 — 网络枢纽 |
| data_engineer | 数据采清洗、特征工程 |
| strategy_researcher | 策略研发、回测、参数优化 |
| risk_analyst | 组合风控、VaR、断路器 |
| execution_agent | 订单执行、模拟券商、订单簿 |
| reporter | 绩效报告、可视化、仪表板 |

**入门指引：**

1. 运行 `/setup` 选择你的 LLM 预设——此将重配全体器灵
2. 运行 `/cpr all` 唤醒休眠之分灵
3. 探索 `strategies/`、`shared_lib/`、`backtests/` 等处之本源
4. Streamlit 仪表板可以 `./scripts/start_dashboard.sh` 启动
